(Vatikan, 15. svibnja 2026.).

> ili pogreške u nijansama značenja. Za autoritativan tekst pogledajte

🇭🇷 **Pročitaj enciklika:** <https://zcuric.github.io/magnifica-humanitas-hr/>

## Što je unutra

Jedna statična HTML datoteka — bez build-koraka, bez vanjskih ovisnosti osim Google Fonts —
koja sadrži cjelovit tekst i sva 224 footnote, plus:

- 📱 **Mobile-first dizajn** s ladicom za navigaciju
- 🎨 **Tipografija**: Fraunces (display) + Newsreader (body) + IBM Plex Sans (UI)
- 🌗 **Svijetla / tamna / auto tema**
- 🔍 **Pretraga teksta** s isticanjem (case- i dijakritički neosjetljiva, Cmd/Ctrl-K)
- 💬 **Footnote popovers** pri prijelazu mišem (klik = sticky)
- 📊 **Statistika dokumenta** (riječi, čitanje, opseg po poglavlju, citirani pape, dokumenti, biblijske knjige, godine)
- 🧘 **Zen / usredotočeni način čitanja** (tipka `Z`)
- 🖨️ **Stilizirano za ispis** (A4, naslovna stranica, fusnote)
- 🌐 **Poveznice na službene prijevode** (EN, IT, ES, FR, DE, PT, PL)
